# Projet ISI - Gestion d'un Cabinet Médical

Projet de développement d'un dossier médical informatisé (Base de données relationnelle et objet-relationnel).

## Membres du groupe
- [ANRAFA SAID]
- [ABDOURAHIM MOHAMED TOIHIR NOURAYAT]

## Structure du projet
- `docs/` : Documentation (MCD, description du projet)
- `sql/` : Scripts de création, triggers de sécurité et requêtes d'analyse
- `src/` : Code source Python
- `tests/` : Journal des jeux d'essais

## Technologies
- **Base de données** : MySQL Workbench
- **Langages** : SQL, Python
- **Outils** : Git/GitHub, Mermaid (pour les schémas)

## Fonctionnalités
- **Sécurité** : Triggers pour valider l'intégrité des dates de visite.
- **Analyse** : Requêtes SQL pour le suivi des visites et la gestion des ordonnances.
- **Automatisation** : Script de connexion en Python pour l'interfaçage avec la base.
